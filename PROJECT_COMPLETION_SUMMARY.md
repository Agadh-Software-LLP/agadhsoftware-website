## 🔧 Customization Guide

### Change Primary Color
Search and replace:
- `blue-900` → your-color-900
- `blue-950` → your-color-950
- `blue-100` → your-color-100
- `blue-50` → your-color-50

### Adjust Spacing (More/Less)
- Sections: Change `py-16 sm:py-24 lg:py-32`
- Cards: Change `p-6 sm:p-8 lg:p-10`
- Buttons: Change `px-6 py-3`

### Update Typography Sizes
- Edit text-4xl, text-5xl, text-6xl values
- Modify font weights (bold, semibold, regular)
- Adjust line-height values
