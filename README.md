# Agadh Software LLP — Company Website

Enterprise-grade services website built with **Next.js (App Router)**, **TypeScript**, and **Tailwind CSS**.

## Goals
- Professional, engineering-driven, trustworthy tone (minimal motion)
- Core message: Agadh Software LLP builds deep, correct, scalable systems and owns them end-to-end
- Pages: Home, About, Services, Industries, Case Study (Kodachadri Chits – CRP Platform), Why Agadh, Tech Stack, Contact

## Development
- Use the VS Code task **dev** to start the dev server.
- Use **build** to produce a production build.
- Use **lint** for linting.

## Notes
- Contact details are placeholders in `src/content/site.ts` (update email/phone when ready). 
- The contact form is client-side with a placeholder submit handler (replace with an API route / email integration later).
